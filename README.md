# Creator Club Backend

A TypeScript NestJS backend API for Creator Club.

## Installation

```bash
npm install
```

## Running the app

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## API Documentation

Once the application is running, visit `http://localhost:3000/api` to view the Swagger documentation.

## Test

```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```

## Features

- TypeScript support
- Swagger API documentation
- Input validation with class-validator
- ESLint and Prettier configuration
- Sample Users module with CRUD operations
